What is flex-direction?

Flex direction allows us to set how we want the information inside of the flex container presented

Options we have are

Row (flex-direction:row)
Column (flex-direction:column)

The same properties can be reversed to read backwards by using

row-reverse (flex-direction:row-reverse)
column-reverse (flex-direction:column-reverse)

Another way to position contents on your page is justify-content
